IVPORT RASPBERRY PI MODULE

IVPort Raspberry Pi Camera Multiplexer

IVPort is the world's first "Raspberry Pi Camera Module" multiplexer board designed for the Raspberry Pi, allowing the connection of multiple camera modules to the Raspberry Pi.

IVPort Camera Multiplexer Module can be directly attached to the GPIO port on the Raspberry Pi. Additionally, multiple IVPort modules (up to 16 cameras) can be stacked on top of each other to increase the number of cameras. It is a module designed for applications that require capturing images from multiple cameras in a short time. With its compact design, it includes 5 Flexible Flat Cable (FFC) connectors. One of these connectors is for the Raspberry Pi input, while the others are used for the camera sensors. Camera selection is controlled via digital outputs on the GPIO port.

FEATURES

  • Compatible with Raspberry Pi and Raspberry Pi Camera Module.

    • It also supports other development boards that can be connected to Raspberry Pi and Raspberry Pi Camera Module.

  • Quick Camera Selection

    • It can switch from one camera to another within 300 milliseconds. This duration may vary depending on the image quality.

  • User-Friendly

    • It can be easily started using Python code examples. You can access the sample codes from here

  • Video Support

    • Camera selection and switching can be done during video recording.
